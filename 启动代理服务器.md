# 启动代理服务器

## 快速解决CORS问题

### 方法1：使用本地代理服务器（推荐）

1. **启动代理服务器**
   ```bash
   cd "/Users/mao/Documents/Coding/Development/Projects/Web/看拼音写词"
   node proxy-server.js
   ```

2. **保持服务器运行**
   - 服务器会在 `http://localhost:3001` 启动
   - 不要关闭这个终端窗口
   - 前端会自动通过代理调用百度API

3. **访问应用**
   - 在浏览器中打开 `index.html`
   - 现在API调用会通过代理，不再有CORS问题

### 方法2：使用浏览器扩展（临时测试）

安装CORS扩展（仅用于开发测试）：
- Chrome: "CORS Unblock" 或 "Allow CORS"
- 启用扩展后再测试

### 方法3：使用服务端中转（生产环境）

如果部署到服务器，可以在服务端调用百度API，前端调用自己的服务。

---

## 注意事项

- 代理服务器只需运行一次，可以一直保持运行
- 如果关闭了代理服务器，应用会提示"代理服务器未运行"
- 代理服务器只转发请求，不会存储任何数据
