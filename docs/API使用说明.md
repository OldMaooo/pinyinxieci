# API使用说明

## 百度AI手写识别API

### 免费额度限制

**每月免费调用次数**：1000次
- 适用于标准版通用手写识别
- 免费额度按月重置，未使用部分不累积到下月
- 超出免费额度后，按实际调用次数计费

**并发限制**：
- 免费额度内单账号最大并发数：**5 QPS**（每秒查询次数）
- 超出并发限制后需升级套餐

### 使用建议

1. **监控使用量**
   - 建议在百度AI控制台定期查看调用量
   - 避免在短时间内大量调用（注意5 QPS限制）

2. **优化调用频率**
   - 当前实现：每次提交答案调用一次
   - 如果练习量大，建议：
     - 控制每日练习次数
     - 或考虑升级套餐

3. **备用方案**
   - 如果免费额度用完，可以考虑：
     - 注册多个百度AI账号（每个账号1000次/月）
     - 升级到付费套餐
     - 切换到其他识别API（腾讯、讯飞等）

### 配置说明

API配置在 `js/config.js` 文件中：
```javascript
baidu: {
    apiKey: 'your_api_key',
    apiSecret: 'your_api_secret'
}
```

### 获取API密钥

1. 访问 [百度AI开放平台](https://ai.baidu.com/)
2. 注册/登录账号
3. 创建应用，选择"文字识别"服务
4. 获取 API Key 和 Secret Key
5. 配置到 `js/config.js` 中
