## 文件说明

### 核心文件
- **index.html**: 单页面应用，包含所有页面模块
- **js/main.js**: 应用入口，路由管理，事件绑定
- **js/storage.js**: 数据存储封装，统一管理LocalStorage

### 功能模块
- **js/handwriting.js**: Canvas手写输入，笔画记录，撤销功能
- **js/recognition.js**: 百度AI识别API调用，图片预处理
- **js/practice.js**: 练习流程控制，计时，自动保存
- **js/errorbook.js**: 错题本管理，按轮/汇总视图
- **js/statistics.js**: 统计计算，结果页展示

### 配置和工具
- **js/config.js**: API配置（密钥、阈值等）
- **js/version.js**: 版本号和更新日志
- **api/baidu-proxy.js**: Vercel Serverless Function代理
- **scripts/extract_pdf.py**: PDF提取工具

### 文档
- **docs/需求文档.md**: 完整需求说明
- **docs/chat.md**: 开发聊天记录总结
- **README.md**: 项目快速开始指南

## 数据存储

所有数据存储在浏览器LocalStorage中：
- `wordbank_data`: 题库数据
- `practice_logs`: 练习记录
- `error_words`: 错题本数据
- `practice_settings`: 用户设置

## 部署

- **前端**: GitHub Pages（静态托管）
- **代理**: Vercel Serverless Function

## 开发工具

- **调试模式**: 导航栏开关，显示调试面板
- **PDF提取**: Python脚本（scripts/extract_pdf.py）
- **本地代理**: Node.js脚本（scripts/proxy-server.js）
