# 看拼音写词 - 生词练习应用

一个帮助小学生通过手写练习掌握生词的网页应用，实现**测试→校对→学习→测试**的自主学习闭环。

## 功能特点

- ✅ **手写输入**：支持iPad手写笔和电脑鼠标输入
- ✅ **自动识别**：使用第三方API进行手写识别
- ✅ **错题记录**：自动记录错题，保存书写快照
- ✅ **直观对比**：并排对比孩子写的字和标准答案
- ✅ **本地存储**：所有数据存储在浏览器本地
- ✅ **导出导入**：支持数据导出和导入，方便换设备使用

## 快速开始

### 1. 打开应用

直接用浏览器打开 `index.html` 即可使用（建议使用Chrome或Safari）。

### 2. 配置识别API（已完成✅）

识别API已配置在 `js/config.js` 文件中，可以直接使用。

如果需要修改配置，编辑 `js/config.js` 文件即可。

### 3. 导入题库

#### 方式一：从PDF提取（推荐）
```bash
# 安装依赖
pip install pdfplumber

# 运行提取脚本
python extract_pdf.py
```

将生成的JSON文件导入系统。

#### 方式二：手动导入JSON

创建JSON文件，格式如下：
```json
{
  "wordBank": [
    {
      "word": "人",
      "pinyin": "rén",
      "grade": 3,
      "semester": "上",
      "unit": 1
    }
  ]
}
```

然后在"题库管理"页面导入。

## 使用说明

### 开始练习

1. 点击"开始练习"
2. 选择题目数量、时间限制和练习范围
3. 点击"开始练习"
4. 根据拼音手写生字
5. 点击"提交"进行识别校验

### 查看错题

1. 在"错题本"页面查看所有错题
2. 可以看到书写快照和标准答案的对比
3. 点击"复习错题"可以重新练习

### 数据管理

- **导出**：在"题库管理"页面点击"导出"，下载JSON文件
- **导入**：选择JSON文件，点击"导入"
- **备份**：建议定期导出数据作为备份

## 技术栈
