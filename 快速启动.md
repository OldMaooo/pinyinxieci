# 快速启动指南

## 解决CORS问题 - 两步搞定

### 第一步：启动代理服务器

**方法1：使用启动脚本（推荐）**
```bash
cd "/Users/mao/Documents/Coding/Development/Projects/Web/看拼音写词"
./start-proxy.sh
```

**方法2：直接运行**
```bash
cd "/Users/mao/Documents/Coding/Development/Projects/Web/看拼音写词"
node proxy-server.js
```

看到 `✅ 代理服务器已启动: http://localhost:3001` 就成功了！

### 第二步：使用应用

1. **保持代理服务器运行**（不要关闭终端窗口）
2. 在浏览器中打开 `index.html`
3. 开始练习，API调用会自动通过代理，没有CORS问题

---

## 说明

- 代理服务器只在本地运行，不会存储任何数据
- 每次使用前需要先启动代理服务器
- 可以一直保持运行，用完再关闭
